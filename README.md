# Console Robot Puzzle - 代码迷宫

一个AI原生人机协同的点击解谜游戏

## 快速开始

### 方法1：直接用浏览器打开（推荐）
1. 双击 `index.html` 文件
2. 游戏会在浏览器中打开
3. 开始游戏！

### 方法2：使用本地服务器（如果方法1不工作）
```bash
# 使用Python启动简单HTTP服务器
python -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000

# 然后在浏览器打开
http://localhost:8000
```

## 游戏玩法

### 目标
逃出诡异的代码房间！有两种通关方式：
1. 找到密码门的4位数密码
2. 破坏密码门（让HP降到0）

### 操作说明
1. **普通模式**：点击物体进行交互
2. **代码撕裂器模式**：点击物体让Alex分析代码
3. **与Alex对话**：在右侧聊天框输入消息
4. **拖拽**：某些物体（如火柴）可以拖拽移动

### 解谜提示
- 🐷 存钱罐可以生成什么？
- 🔥 火柴可以与什么互动？
- 📄 信纸上藏着什么秘密？
- 🤖 让Alex帮你分析和修改代码！

## 技术信息

- **图形引擎**：p5.js
- **AI助手**：DeepSeek API
- **CORS代理**：corsproxy.io

## 故障排除

### 如果Alex无法回复
- 检查网络连接
- 确保CORS代理服务可用
- 查看浏览器控制台的错误信息

### 如果游戏画面不显示
- 确保使用现代浏览器（Chrome、Firefox、Edge等）
- 检查是否加载了p5.js库

## 游戏物体说明

| 物体 | 功能 | 代码权限 |
|------|------|----------|
| 密码门 | 输入密码通关 | 全部权限1（不可读）|
| 存钱罐 | 点击生成硬币 | onClick权限4（可编辑）|
| 信纸 | 火柴碰撞显示文字 | onCollide权限1 |
| 火柴 | 可拖拽 | onDrag权限3（可读）|
| 陀螺 | 点击旋转90度 | onClick权限4 |
| 硬币 | 向下掉落碰撞 | 自动生成 |
